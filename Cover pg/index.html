GREEN COVER DETECTOR

Green Cover Detector calculates the green part
and building part in Satellite maps. Provides comparision
between both areas to detect if area is green certified or not.

How it works

Green Cover Detector is designed to calculate green area in any part of the world.
Using Google Earth Engine and Numpy with Python, Satellite Imagery
from Landsta-8 and Copernicus and geojson.io.
Select Map in Get Started button and map will pop-up.
Select desired co-ordinates and select Submit.
You'll find Green Area, Built Area, Green Certification and Recommendations in new window.
You can use the same technique to find Green Area in any part of the world.
